Navigating to eCitations

There are two ways to navigate to the eCitations section of the application:

  1. Using the keyboard shortcut Ctrl + E, or
  2. Click on the Quick Menu Table button and then click on eCitations

    3. Using the Quick Menu Table button

Downloading new eCitations

New eCitations must be downloaded into the software before they can be referenced.

The eCitations screen. Notice the download button at the top right.

By simply clicking the Download button, any eCitation that has not been previously been downloaded from the eCitation Database in Tuscaloosa will be downloaded to the software. The software also inserts the Defendant Name, Charge, and Officer in the ‘Local Case Info’. These fields are used to later create records in the Case Table.

The Download button

When finished successfully the message at the bottom will say 'Process Completed!'

Indication of a completed eCitation download

eCitation Reports

There are four types of eCitation reports that can be generated: eCitation List Report, eCiation Swear Report, eCitation Create Cases, and eCitation Tickets.

eCitation List Report

Run the eCitation List Report to see what tickets have been downloaded and have not yet had cases created from them. Print > eCitation > eCitation List Report

Navigating to the eCitation List Report

An example of an eCitation List Report

This report shows 4 tickets were downloaded without any errors.

eCitation Swear Report

The eCitation Swear Report is optionally run after the eCitation List Report. If the Officer has already turned in a Transmittal Sheet, this report may not be necessary. Print > eCitation > eCitation Swear Report

The window prior to running the report allows for several different options on which Officer’s tickets are to be placed on a Transmittal Sheet.

eCitation Swear Report options

The four tickets downloaded are ready to be placed on a transmittal form. The check boxes next to the tickets allow them to be included or excluded.

Selecting a particular officer in the Officer Dropdown will display only that officer’s citations.

Selecting only a specific officer's eCitations

‘Include All Officers’ will allow Tickets for Officers who are no longer active.

‘Mark as Sworn’ is rarely used. Tickets are considered sworn to when cases are created with them. Tickets may be marked as sworn to prior to that time if desired.

‘Include Citations Already Processed’ allows for tickets to be placed on a Transmittal form that have already had cases created from them.

The Transmittal Form generates one page per officer.

Example of a UTC Transmittal form

With a signature pad, the Officer may sign the Transmittal.

eCitation Create Cases

Now that the eCitations have been downloaded and officers have sworn to them cases may be created. Print > eCitation > eCitation Create Cases.

eCitation Create Cases options

A particular Officers cases may be specified in the Officer Dropdown.

‘Print eCitations’ determines whether or not the actual eCitations are printed out.

Created eCitation cases report

The Case Records are now created in the Cases Table. The Resulting Case Number is listed in the last column of the report.

The actual eCitation is printed with the new Case Number on it. The document is then attached to the eCitation Record and the Case Record.

An eCitation with its associated case number

With a signature pad the Officer and the Magistrate may sign the eCitation.

All the tickets may be signed at once by holding the Ctrl Key down while clicking on the signature field.

A judge may later fill in the fields on the right side and sign it as well.

eCitation Tickets

The last eCitation Report allows tickets to be reprinted or printed for the first time if they were not previously printed in ‘eCitation Create Cases’. Print > eCitation > eCitation Tickets
