Adding a New Charge

Occasionally you may get a new Traffic offense, Misdemeanor, or Local Ordinance. These will need to be added as new records in Charges before they can be used.

Steps

  1. Open the Charges table in the Quick Menu (green table)

    Quick Menu button

    Selecting Charges in the Quick Menu

  2. Click on the ‘Add Record’ icon in the upper left corner

    Add Record button

  3. Type in the charge

    Charge screen

  4. For Traffic Cases, Type in the Charge Code and State Charge Code. The Charge Code is not Required for Misdemeanors and Local Ordinances.

    Entering the Charge code

    This Charge Code or ‘TCode’ may be found on the issued eCitation.

    Getting Charge code from TCode on eCitation

  5. Enter the State Statute for this Charge. Traffic Tickets will normally start with "32-…" and Misdemeanors will often start with "13A-…"

    Entering the State statute number

  6. Enter the Local Ordinance that adopts all the State Codes.

    Entering the Local Ordinance code

  7. If the New Charge is not a State Charge, but just a Local Ordinance, leave the Statute Blank and enter the specific Local Ordinance Number for that Charge.

    Local Ordinance Number without State charge

  8. If the charge is payable and NO Appearance is Required, enter the amount that the defendant would pay if they Failed to Appear in 'FTA Amount'. In 'FTA Add' enter the amount the fine was increased. In this example, RUNNING A STOP SIGN could be paid prior to the court date for $191.00. If the Defendant did not appear that amount would be increased by $25 to $206.

  9. If Appearance is Required, the Appearance Required box is checked and the 'FTA Amount' should be blank.

    Checking "Appearance Required"

  10. Optionally enter a Note as a Reminder for the Docket. Anytime this charge appears on the docket, this Note would also be printed.

  11. Check any boxes for which the Charge falls in the category.

  12. Optionally enter the Maximum Fine Amount and Jail Time for use on the Guilty Plea Waivers.

  13. Complaint Verbiage may optionally be filled in. This verbiage would be used as a starting point for a new complaint.

    The Complaint Verbiage which appears on new cases with this charge

  14. Finally, save the newly entered Charge by clicking the Save Record button.

    The Save Record button
