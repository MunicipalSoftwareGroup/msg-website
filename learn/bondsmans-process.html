Bondsman's Process

Steps

  1. In the Bond Surety Forms menu, selecting ‘Application for Bondsmans Process (CR-28)’ will display the following screen.

  2. In the Bond Field select the appropriate Surety Bond.

    Selecting the appropriate Surety Bond.

  3. Click on ‘OK’ to Print the Form.

    Clicking the 'OK' button

    The Application Form

  4. After the Application is accepted, the actual Bondsman’s Process (CR20) is Printed.

    The Bondsman's Process Form

    After five days or in certain events, a Notice to Return Bondsman’s Process (CR-30) may be issued.

    Notice to Return Bondsman’s Process (CR-30)

    Failure of the Surety to return the process may require a CR29 ‘Failure to Return Bondsman’s Process’ to be issued.

    Failure to Return Bondsman’s Process Notice

As each of the forms are printed the appropriate dates are placed in the correct fields on the second page of the Surety Bond Record.

Surtety Bond Record's Bondsman's Process dates
