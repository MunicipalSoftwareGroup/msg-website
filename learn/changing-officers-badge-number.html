Changing and Officer's Badge Number

If an officer's badge number changes, it can be updated in the associated Officer record.

Steps

  1. Navigate to the Officers table by clicking Tables > Parties > Officers.

    Navigating to the Officers table

  2. Click on the Search button at the top and find the officer.

    The Search button

    Searching for the officer by name

  3. Click on the Badge number and change it to the correct number.

    Changing the badge number

  4. Click on the Save icon

    The Save button

There should not be any duplicate badge numbers, and there should only be one record for any one officer.

Badge Numbers belonging to inactive officers may have a letter added to them in order to make them available for use (Example: 00830 changes to 0830B).

An inactive officer's record may be deleted without affecting any of the officer's cases.
